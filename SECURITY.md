# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please email
[security@odysseytherapeia.com](mailto:security@odysseytherapeia.com).

Please do not open a public GitHub issue for security reports.

## Response Time

We aim to acknowledge and respond to vulnerability reports within 7 business
days.

## Supported Versions

We currently support only the latest release on the `main` branch.

## Scope

This repository is a game simulator rather than production infrastructure, but
we take all security reports seriously and will review them in good faith.
